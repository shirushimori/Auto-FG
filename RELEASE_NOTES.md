## What's included

### Windows
- **setup_x64.zip** — for 64-bit Windows PCs
- **setup_arm64.zip** — for ARM64 Windows (Surface, etc.)

### Linux
- **Auto-FG-x86_64-linux.tar.gz** — for 64-bit Linux

Each archive contains the compiled binaries, setup script, and README.

### How to use

**Windows:** Extract the ZIP, run `setup.bat` or launch `Ffast-auto-downloader.exe` directly.

**Linux:** Extract the tarball, run `./setup.sh` or launch `./Ffast-auto-downloader` directly.
